## 🚀 使用方式

1. **访问首页**：`http://localhost:8080/index.html`
2. **商品浏览**：点击"商品列表" → 选择商品 → 查看详情
3. **订单管理**：点击"订单管理" → 查看订单列表和详情
4. **支付测试**：点击"支付中心" → 创建订单 → 支付
5. **系统测试**：点击"系统测试" → 测试各种API功能
