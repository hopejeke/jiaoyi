# IDEA 快速运行指南

## 问题：看不到绿色运行按钮？

### 解决方法1：让IDEA识别Maven项目

1. **右键点击 `pom.xml` 文件**
2. **选择 "Add as Maven Project"** 或 **"Add as Maven Project"**
3. 等待IDEA下载依赖（右下角会显示进度）
4. 完成后，`main` 方法旁边就会出现绿色运行按钮 ▶️

### 解决方法2：手动创建运行配置

如果还是没有绿色按钮，手动创建运行配置：

1. **点击右上角的运行配置下拉框**（显示 "Add Configuration..." 或类似文字）
2. **点击 "Edit Configurations..."**
3. **点击左上角的 `+` 号**
4. **选择 "Application"**
5. **配置如下：**
   - **Name**: `LoadTestMain`
   - **Main class**: 点击右侧的文件夹图标，选择 `com.jiaoyi.loadtest.LoadTestMain`
   - **Module**: 选择 `load-test`（如果看不到，先执行方法1）
   - **Program arguments**: （可选）留空使用默认配置，或填入：
     ```
     --url http://localhost:8080 --threads 20 --duration 120
     ```
6. **点击 "OK"**
7. **点击右上角的绿色运行按钮** ▶️

### 解决方法3：使用快捷键

1. 将光标放在 `main` 方法内部（第21行）
2. 按快捷键：**`Shift + F10`**（Windows/Linux）或 **`Ctrl + R`**（Mac）
3. 或者按：**`Ctrl + Shift + F10`**（Windows/Linux）

### 解决方法4：右键菜单

1. 将光标放在 `main` 方法内部
2. **右键点击**
3. 选择 **"Run 'LoadTestMain.main()'"**

## 如果还是不行

### 检查项目结构

确保目录结构如下：
```
scripts/load-test-java/
├── pom.xml
└── src/
    └── main/
        └── java/
            └── com/
                └── jiaoyi/
                    └── loadtest/
                        ├── LoadTestMain.java
                        ├── LoadTester.java
                        └── LoadTestStats.java
```

### 检查Java版本

1. **File → Project Structure → Project**
2. 确保 **Project SDK** 是 Java 11 或更高版本
3. 确保 **Project language level** 是 11 或更高

### 重新导入项目

1. **File → Invalidate Caches / Restart...**
2. 选择 **"Invalidate and Restart"**
3. 重启后，右键 `pom.xml` → **"Add as Maven Project"**

## 最简单的运行方式

如果以上都不行，直接用命令行：

1. 打开IDEA的Terminal（底部工具栏）
2. 执行：
   ```bash
   cd scripts/load-test-java
   mvn clean compile exec:java -Dexec.mainClass="com.jiaoyi.loadtest.LoadTestMain"
   ```

## 常见问题

**Q: 提示找不到类？**
A: 右键 `pom.xml` → "Add as Maven Project"，等待依赖下载完成

**Q: 提示找不到依赖？**
A: 在IDEA右侧打开Maven工具窗口，点击刷新按钮，或运行 `mvn clean install`

**Q: 编译错误？**
A: 检查Java版本，确保是Java 11+
